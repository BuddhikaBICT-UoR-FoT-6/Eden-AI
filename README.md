# Eden AI 🌴

Eden AI is an enterprise-grade portfolio project designed as a vibe-based hotel and villa finder for the Sri Lankan tourism market. 

By leveraging the power of Generative AI, Eden AI allows travelers to find accommodations not just by location or price, but by the "vibe" and atmosphere they are seeking (e.g., *jungle luxury*, *surf-chill*, *colonial charm*).

## Tech Stack
- **Frontend:** React (TypeScript) with Tailwind CSS
- **Backend:** Spring Boot (Java 17)
- **Database:** PostgreSQL
- **AI Layer:** Google Gemini API (Structured Outputs)

## Project Structure
- `/frontend` - React client application
- `/backend` - Spring Boot REST API
